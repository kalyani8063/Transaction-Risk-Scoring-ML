# Transaction Risk Scoring using Machine Learning

## Overview
This project implements a machine learning system that assigns risk scores to financial transactions. The goal is to identify potentially suspicious or fraudulent transactions based on transaction attributes.

The model analyzes features such as transaction amount, time, location, and merchant category to generate a probabilistic risk score between 0 and 1.

## Motivation
Financial institutions rely on automated systems to monitor transactions and detect fraudulent activities. This project demonstrates how machine learning can be used to build a basic transaction risk scoring system similar to those used in fintech applications.

## Features
- Data preprocessing and feature engineering
- Transaction risk scoring using machine learning
- Classification models for fraud detection
- Risk score generation for new transactions
- Data visualization and exploratory analysis
